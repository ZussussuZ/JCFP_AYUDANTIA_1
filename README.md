
# Materiales:

X2 LEDS ROJOS
X2 LEDS AZUL
X2 LEDS VERDE

X1 RESISTENCIA DE 220 OHM

X1 PULSADOR
X1 PLACA ARDUINO
X1 PROTOBOARD

CABLES PARA PROTOBOARD
USBB A USB 2.0


# PASOS DE ARMADO:

Instalamos los leds, boton y Resistencia
ubicamos los cables como se ve en el diagrama
copiamos y pegamos el codigo en Arduino IDE y compilar y probar
